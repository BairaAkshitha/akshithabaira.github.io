Extending DoseDiff for Head & Neck Radiotherapy

Distance-aware diffusion for 3D radiotherapy dose prediction ¬∑ Medical AI ¬∑ PyTorch

This project extends the DoseDiff diffusion model to handle 3D dose prediction for head and neck radiotherapy using the OpenKBP dataset. I explored distance-aware conditioning mechanisms and denoising schedules to stabilize training and improve dose distribution accuracy.

The work enhances treatment planning by producing more consistent dose maps that respect anatomical distances and critical structure constraints, contributing to safer and more effective radiotherapy planning.

Document Summarizer (Transformers + FastAPI)

Multi-document text summarization with transformer models and an API layer.

I built a document summarization system using Hugging Face transformer models to generate concise summaries for long-form text such as research papers, business documents, and domain-specific content.

The system supports multi-document inputs, handles pre-processing and chunking, and exposes a FastAPI endpoint for easy integration into applications or dashboards. This improves readability, speeds up review cycles, and helps users extract key insights quickly.

Brain Tumor Segmentation in MRI (UNet)

Deep learning-based semantic segmentation for tumor localization in MRI scans.

This project implements a UNet-based architecture for segmenting brain tumors from MRI scans. The model learns to distinguish tumor regions from healthy tissue, producing segmentation masks that can be used to support radiology workflows and pre-treatment planning.

The pipeline includes data augmentation, preprocessing, and post-processing steps to handle MRI variability and to improve the robustness and clarity of the segmentation outputs.

UNet Semantic Segmentation Medical AI

Canine VHS Prediction using CNN

Veterinary radiology model for vertebral heart size (VHS) estimation.

In this veterinary-focused project, I developed a convolutional neural network to estimate the vertebral heart size (VHS) from canine thoracic radiographs. VHS is an important metric for assessing cardiac enlargement and potential heart disease in dogs.

The system outputs VHS predictions that can assist veterinarians with earlier detection of cardiac abnormalities, helping to prioritize further diagnostics and treatment decisions.
